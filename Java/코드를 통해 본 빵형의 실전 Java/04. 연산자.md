# 04. 연산자
* 연산자 : 어떤 연산을 할지를 나타내는 가감승제
* 피연산자 : 연산의 대상
<br/>

## 4.1 산술연산자
```
+, -, *, /, %(나머지가 나옴)
```
<br/>

## 4.2 증감 연산자
```
- 전치 연산자 : 연산자가 앞에
- 후치 연산자 : 연산자가 뒤에
```
<br/>

## 4.3 비교 연산자
```
두 개의 변수나 리터럴을 비교하는 연산
비교 조건이 맞으면 true, 틀리면 false
```
<br/>

## 4.4 논리 연산자
```
두 개의 피연산자를 대상으로 연산을 수행하여 참과 거짓을 결정할 때 사용
= 조건식
```
<br/>

## 4.5 비트 연산자
```
&, |, ^, ~, <<, >>, >>>
```
<br/>

## 4.6 복합대입 연산자
```
+=, -=, *=, /=, %=
&=, !=, ^=, <<=, >>=, >>>
```
<br/>

## 4.7 삼항 연산자
```
( 조건식 ) ? 참일때 : 거짓일때
```

<br/>
